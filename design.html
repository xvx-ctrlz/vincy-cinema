Overall Design of the website


From the above sketch diagram, it has shown the overall structure of the web pages in my project.
In each webpages, the basic elements included are as follow:

  1. A Menu that is hoverable and designed with transparency of 0.7
  2. The logo of the cinema with animation effect
  3. A line that have shown the cinema's name "Chaplin's Cinema" and the message to the customer with marquee effect
  4. The part under the above three elements would be the main content

The usage of style sheets and groups of styles


Five style sheets are used in the project. Since putting all style in an external style sheet is more conivient for editing, all css coding are only included in the style sheets and no style is embedded in the html files.

  1. ticket.css

    In "ticket.css", it is used to arrange the specific layout for "ticket.html". For example, the form for the users to input the movie tickets they would like to purchase.

  2. design.css

    In "design.css", it is used for arranging the layout of the design page.

  3. now.css

    In "now.css", it is used to format the layout for the "now.html", which included the information of the movies and let the users to choose from.

  4. print.css

    In "print.css", it is used to format the printing of the ticket booked by the customers.

  5. styles

    In "style.css", it is used to format the general and common layout of the four web pages, which are "index.html","now.html","print.html" and "ticket.html".
